              When you Run Webcam, sensor_runner starts python -m weapon_ai.webcam_layer8_runner
              (cwd layer4_inference/), which loads and runs weapon_ai/infer_thermal_objects.py (same logic as
              python -m weapon_ai.infer_thermal_objects). weapon_yolo_model (e.g. yolov8n.pt) is the
              person / object detector (--conf = weapon_conf, min box = weapon_min_box_px).
              weapon_checkpoint (e.g. gun_prob_best.pt) scores each person crop for p(gun) / safe vs unsafe
              (weapon_unsafe_threshold / weapon_gun_threshold). Optional orange boxes: separate firearm YOLO via weapon_gun_*.
              Keys live under webcam in JSON; most flags are forwarded as --weapon-extra-args on that subprocess.
              layer8_ui.inference_overlay is only a server-side stub today (not the live subprocess).
